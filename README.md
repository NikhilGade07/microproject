# 📚 Classroom Lite

A lightweight Learning Management System (LMS) for teachers and students — built as a micro project.

🔗 **Live Demo:** [classroom-management-system-production-99c7.up.railway.app](https://classroom-management-system-production-99c7.up.railway.app/)

---

## ✨ Features

### 👨‍🏫 Teacher
- Create and manage courses
- Post announcements
- Add assignments with deadlines and marks
- View and grade student submissions
- Export submissions as CSV

### 👨‍🎓 Student
- Join courses using a course code
- View announcements and assignments
- Submit assignments
- Track grades and feedback

---
